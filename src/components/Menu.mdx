# Menu

Sidebar Menu Component

## Properties

<PropsTable of={Menu} />

### items

items can have the following properties:
<ul>
    <li>
        name - name of the menu item (required)
    </li>
    <li>
        href - link of the menu item
    </li>
    <li>
        featherIcon - feather icon name
    </li>
    <li>
        badge - badge class name (bootstrap names)
    </li>
    <li>
        badgeText - text of the badge
    </li>
    <li>
        children - sub menu items
    </li>
</ul>
