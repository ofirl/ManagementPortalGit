# Animated Component

HOC for animating elements/components

the underlying element/component will recieve 2 additional props and will need to add them to the render:
<ul>
    <li> 
        animationClassName - needs to be added to the className attribute of the rendered element
    </li>
    <li> 
        animationStyle - needs to be added to the style attribute of the rendered element    
    </li>
</ul>
